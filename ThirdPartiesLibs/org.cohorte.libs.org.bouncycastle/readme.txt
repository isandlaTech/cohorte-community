LATEST JAVA RELEASES
============================================================
https://www.bouncycastle.org/latest_releases.html
Release 1.68 is now available for download.
This release is a primarily about TLS and to alert people to a recent CVE. Concerning TLS, the BCJSSE now supports TLS 1.3 and session resumption for TLS 1.2 and earlier. 
A few small bugs in the ASN.1 library and PGP package have also been fixed and the PGPSignatureSubpacketGenerator now supports the editing of a pre-existing signature sub-packet list.
